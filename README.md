# 该仓库为react项目的模版仓库

## 基础依赖包说明

- node版本 => 18.12.0
- react版本 18.x
- react-router 6.x
- typescript 5.x

## 状态管理

- 建议使用mobx，如需要其他的状态管理库，请删除package.json中的依赖 (mobx, mobx-react-lite)

## 样式

- style-components
- less

## 开发规范

```
建议安装vsCode相关插件 Prettier-Code 、 Eslint，以便有更好的开发体验
```

- 集成了eslint
- 集成了husky
- 集成了prettier
